## Sintaxis

```js
add_editor_style( $HojaDeEstilo )
```

## Parámetros

```js
$HojaDeEstilo: ruta de la hoja de estilo que deseamos importar al pluging
```

## Ejemplo

```js
function add_new_style() {
    add_editor_style( 'file-name-here.css' );
}
add_action( 'admin_init', 'add_new_style' );
```
